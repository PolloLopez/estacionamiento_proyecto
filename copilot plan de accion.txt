Login / Logout

login_view implementa autenticación manual, lo cual rompe compatibilidad con middlewares y futuros logins externos (Google).

logout_view borra la sesión pero no usa django.contrib.auth.logout.

Corrección:

python
from django.contrib.auth import authenticate, login, logout

def login_view(request):
    if request.method == "POST":
        correo = request.POST.get("username")
        password = request.POST.get("password")
        usuario = authenticate(request, correo=correo, password=password)
        if usuario:
            login(request, usuario)
            return redirect("inicio")
        return render(request, "usuarios/login.html", {"form": {"errors": True}})
    return render(request, "usuarios/login.html")

def logout_view(request):
    logout(request)
    return redirect("login")

Duración de estacionamiento

Estás usando Decimal(duracion) y luego % 1 != 0. Eso puede romper porque Decimal no soporta % como esperás.

Corrección: parsear a int si son horas enteras, o usar Decimal(duracion) * 2 % 1 si querés medias horas.
